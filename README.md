# TimeTracker

This application allows you to track the time you spend on various activities.

## Screenshot

![Screenshot](./img/screenshot.PNG)

## Technologies
* React
* JavaScript
* Redux
* Bootstrap
* SASS

## Current features

- Start/stop timers
- Create new timers
- Edit timers
- Delete timers
- Save timers in local storage
